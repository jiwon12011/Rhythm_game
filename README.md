# 마요나카☆아쿠아비트 (MIDNIGHT AQUA BEAT)

> 밤 12시, 폐수족관의 수조에 다시 불이 켜진다 — 물고기 대신, 소녀들이 빛난다.

가로 16:9 웹 리듬게임 × 캐릭터 수집. 수집한 밴드 멤버의 컨셉포토가 라이브 배경이 되고, 콤보가 쌓일수록 배경이 밝아지는 "덕질 보상형" 리듬게임.

## 에셋

캐릭터 이미지는 별도 생성 후 `assets/` 하위에 투입한다. 파일명 규칙과 규격은 `SPEC.md` §12 참고. 에셋이 없어도 게임은 폴백 디자인으로 동작해야 한다.

```
assets/
├── photos/   # 컨셉포토 {id}_{n|r|e}.webp (2560×1440)
├── chars/    # 전신 {id}_full.png · 표정 {id}_face_{joy|fire|comfort}.png
├── covers/   # 앨범아트 {songId}.webp (1024×1024)
└── logos/    # 밴드 로고 {bandId}.png
```
